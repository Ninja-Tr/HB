Hadron HDX7512 USB-RS232 Çevirici Kablo


 Bu ürün sayesinde seriport'u olmayan bilgisayarlarınıza usb çıkışını kullanarak seriport(comport) ekleyebilirsiniz.

Neler yapılır?
Programlayıcınızı seriport'u olmayan laptop'unuzda kullanabilirsiniz.
Reciever yazılımı yükleyebilirsiniz ve comport ile yapılabilen herşeyi yapabilirsiniz.....
Bu ürünün diğerlerinden farkı Rs232 protunun tüm özelliklerini kullanabilmenizi sağlamaktır....

Özellikler :
- USB 2.0
- Seri tipi cihazların USB portuna bağlantısını sağlar.
- Kablolu USB Erkek Fiş, RS232 Erkek Fiş
- Windows 98, Windows ME, Windows 2000, Windows XP, Mac OS 8.6 ve üstü işletim sistemlerini destekler.
 
Bu Ürün ile Uydu Alıcısına yazılım yüklemek için RS 232 Kablo extra olarak alınmalıdır.
